# 자바스크립트의 구성
	
	코어(ECMAScript) + 문서객체모델(DOM)  + 브라우저객체모델(BOM)
	

# 크롬에서 자바스크립트 오류 검색

		크롬 > 맨 오른쪽 아이콘(맞춤선택및 제어) > 도구 더보기 > 확장 프로그램
		
		검색 : javascript error notifier
		
		검색 : js runtime inspector (월리의 기능을 모르겠음)
